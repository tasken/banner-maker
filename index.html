DS banner maker

Create custom icons and menu titles for Nintendo DS homebrew ROMs.

Icon specs

DS icons are 32 × 32 px with only 16 colors. Color at index 0 is reserved for transparency. This tool downscales and quantizes any image automatically, but starting with a matching image gives sharper results.
